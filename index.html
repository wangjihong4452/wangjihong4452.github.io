❓ 常见问题 FAQ

1. 如何使用远程控制精灵？

首先，请在目标设备的屏幕上找到远程控制精灵的二维码。然后，在您的电脑或手机上打开远程控制精灵客户端，扫描该二维码将其添加到远程列表。添加成功后，即可发起远程连接。

2. 计算器功能支持哪些运算？

基础计算器支持加、减、乘、除四则运算。在“扩展功能区”的科学计算器中，您可以执行更复杂的数学运算，如三角函数 (sin, cos)、对数 (log) 和指数等。

3. 工作清单的数据会丢失吗？

不会。您的工作清单数据会保存在浏览器的本地存储 (LocalStorage) 中。只要您不清除浏览器缓存，数据就会一直保留，即使关闭浏览器再重新打开也是如此。

4. 天气功能为什么显示“暂无天气数据”？

当前版本的天气功能是一个占位符，需要接入第三方天气 API 才能显示真实数据。我们计划在未来的更新中完善此功能。

5. 如何反馈问题或建议？

由于这是一个静态网页，暂时没有内置的反馈系统。您可以通过您获得此链接的渠道向我们反馈，我们会认真考虑每一条建议。
